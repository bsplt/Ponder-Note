<feature>
  <name>Todo Grouping by Tags with Recency Ordering</name>
  <files>app/src/utils/todoGrouping.ts, app/src/utils/todoGrouping.test.ts</files>
  <behavior>
Group todos by parent note tags with cross-group appearance and activity-based ordering.

**Types:**
```typescript
type TodoItem = {
  text: string
  checked: boolean
  noteStem: string
  lineNumber: number
  charOffset: number
}

type NoteSummary = {
  stem: string
  title: string
  createdAt: number
  updatedAt: number | null
  tags: string[]
}

type TodoGroup = {
  tag: string
  todos: TodoItem[]
  mostRecentEdit: number  // For group ordering
}

function groupTodosByTags(
  todos: TodoItem[],
  notes: NoteSummary[]
): TodoGroup[]
```

**Test Cases:**

1. **Single tag note:**
   - Note A: tags=["work"], updatedAt=1000, todos=["task1", "task2"]
   - Output: [{ tag: "work", todos: [task1, task2], mostRecentEdit: 1000 }]

2. **Multi-tag note (cross-group appearance):**
   - Note A: tags=["work", "urgent"], updatedAt=1000, todos=["task1"]
   - Output: [
       { tag: "work", todos: [task1], mostRecentEdit: 1000 },
       { tag: "urgent", todos: [task1], mostRecentEdit: 1000 }
     ]

3. **Untagged note:**
   - Note A: tags=[], updatedAt=1000, todos=["task1"]
   - Output: [{ tag: "Untagged", todos: [task1], mostRecentEdit: 1000 }]

4. **Group ordering by recent activity:**
   - Note A: tags=["work"], updatedAt=1000, todos=["task1"]
   - Note B: tags=["personal"], updatedAt=2000, todos=["task2"]
   - Output: [
       { tag: "personal", todos: [task2], mostRecentEdit: 2000 },
       { tag: "work", todos: [task1], mostRecentEdit: 1000 }
     ] (personal first, more recent)

5. **Within-group ordering by recency:**
   - Note A: tags=["work"], updatedAt=1000, todos=["taskA"]
   - Note B: tags=["work"], updatedAt=2000, todos=["taskB"]
   - Output: [{ tag: "work", todos: [taskB, taskA], mostRecentEdit: 2000 }]

6. **Untagged group at bottom:**
   - Note A: tags=["work"], updatedAt=2000, todos=["task1"]
   - Note B: tags=[], updatedAt=3000, todos=["task2"]
   - Output: [
       { tag: "work", ... },
       { tag: "Untagged", ... }
     ] (Untagged always last)

7. **Use createdAt fallback if updatedAt null:**
   - Note A: tags=["work"], createdAt=1000, updatedAt=null, todos=["task1"]
   - Output: mostRecentEdit should be 1000 (from createdAt)

8. **Empty todos array:**
   - Input: todos=[], notes=[...]
   - Output: []

9. **Todo from note not in notes array (orphan):**
   - todos=[{ noteStem: "missing", ... }], notes=[]
   - Output: Skip orphan todo (not in any group)

10. **Sort stability (same updatedAt):**
    - Note A: tags=["work"], updatedAt=1000, title="Alpha", todos=["taskA"]
    - Note B: tags=["work"], updatedAt=1000, title="Beta", todos=["taskB"]
    - Output: Within group, alphabetical by title (taskA before taskB)

**Algorithm:**
1. Build note lookup map: stem -> NoteSummary
2. Create tag -> todos map
3. For each todo, find parent note
4. For each tag in note.tags (or "Untagged" if empty):
   - Add todo to tag group
   - Track most recent edit in group
5. Sort groups by mostRecentEdit descending, "Untagged" always last
6. Within each group, sort todos by note recency then title
