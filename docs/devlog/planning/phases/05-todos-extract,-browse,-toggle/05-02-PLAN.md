  <behavior>
Toggle checkbox at specific line + character offset with verification before write.

**Test Cases:**

1. **Toggle unchecked to checked:**
   - Input: body="- [ ] task", line_number=0, char_offset=2
   - Output: Ok(true), body now "- [x] task"

2. **Toggle checked to unchecked:**
   - Input: body="- [x] task", line_number=0, char_offset=2
   - Output: Ok(false), body now "- [ ] task"

3. **Toggle uppercase [X] to unchecked:**
   - Input: body="- [X] task", line_number=0, char_offset=2
   - Output: Ok(false), body now "- [ ] task"

4. **Error: line number out of bounds:**
   - Input: body="- [ ] task" (1 line), line_number=5
   - Output: Err(ToggleError::CheckboxNotFound)

5. **Error: char offset out of bounds:**
   - Input: body="- [ ] task", line_number=0, char_offset=100
   - Output: Err(ToggleError::CheckboxNotFound)

6. **Error: no checkbox at position:**
   - Input: body="plain text", line_number=0, char_offset=0
   - Output: Err(ToggleError::CheckboxNotFound)

7. **Error: wrong pattern at position:**
   - Input: body="- [?] malformed", line_number=0, char_offset=2
   - Output: Err(ToggleError::CheckboxNotFound)

8. **Multiline note toggle:**
   - Input: body="# Title\n- [ ] todo\nMore text", line_number=1, char_offset=2
   - Output: Ok(true), line 1 changed to "- [x] todo", other lines unchanged

9. **Preserve indentation:**
   - Input: body="  - [ ] indented", line_number=0, char_offset=4
   - Output: Ok(true), becomes "  - [x] indented" (spaces preserved)

10. **Preserve surrounding text:**
    - Input: body="prefix - [ ] task suffix", line_number=0, char_offset=9
    - Output: Ok(true), becomes "prefix - [x] task suffix"

**Algorithm:**
1. Split body into lines
2. Verify line_number < lines.len()
3. Verify char_offset + 3 <= line.len()
4. Extract 3-char substring at offset
5. Match against "[ ]", "[x]", "[X]"
6. If no match, return Err(CheckboxNotFound)
7. Determine new_state: "[ ]" -> "[x]", "[x]"|"[X]" -> "[ ]"
8. Build new line with replaced checkbox
9. Join lines with \n
10. Return Ok((new_body, new_state == "[x]"))
