Create `Editor.tsx` with a textarea-based editor that loads note content via `noteRead` and focuses on mount.
Implement cursor placement: new notes place the cursor after `# `; existing notes place it at position 0.
Autosave with a short debounce (e.g., 400-600ms) using `noteSave` and track `lastSavedAt` on success.

If exiting a new note whose body is still `# ` (or `#` with only trailing newlines), call `noteDiscard` and skip saving.
Use native textarea undo/redo (no custom stack).
