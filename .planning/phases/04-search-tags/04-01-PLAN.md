  <action>
Create Rust command `workspace_update_note_tags(stem: String, tags: Vec<String>)` that:
1. Loads the note sidecar at `.ponder/meta/{stem}.json`
2. Updates the `tags` field with the provided array (replacing existing tags)
3. Writes the updated sidecar back to disk atomically (write to temp, rename)
4. Returns CommandResult<()> (ok with empty value on success, error on failure)

Implementation notes:
- Use existing sidecar read/write patterns from workspace_service.rs
- Trim whitespace from each tag string and filter out empty strings before saving
- Sort tags alphabetically before saving for consistent sidecar file diffs
- If sidecar doesn't exist, return error "note_not_found"
- If the note Markdown file doesn't exist, return error "note_not_found"

Register the command in `lib.rs` invoke_handler.

Then add TypeScript wrapper in `app/src/api/workspace.ts`:
```typescript
export async function workspaceUpdateNoteTags(stem: string, tags: string[]): Promise<void> {
  const res = await invoke<CommandResult<void>>('workspace_update_note_tags', { stem, tags })
  return unwrap(res)
}
```

Create Rust command `workspace_get_all_tags()` that:
1. Lists all note sidecars under `.ponder/meta/` in the active workspace
2. Loads each sidecar's `tags` field (array of strings)
3. Collects all unique tags across all notes into a Vec<String>
4. Sorts tags alphabetically (case-insensitive sort)
5. Returns CommandResult<Vec<String>> with the unique tag list

Implementation notes:
- Use existing workspace service patterns for listing sidecars (similar to workspace_list_notes)
- Skip sidecars with missing or invalid tags fields (don't fail the entire command)
- Empty tags arrays are fine (just don't contribute to the result)
- If no workspace is active, return error "no_workspace"

Register the command in `lib.rs` invoke_handler.

Then add TypeScript wrapper in `app/src/api/workspace.ts`:
```typescript
export async function workspaceGetAllTags(): Promise<string[]> {
  const res = await invoke<CommandResult<string[]>>('workspace_get_all_tags')
  return unwrap(res)
}
```
