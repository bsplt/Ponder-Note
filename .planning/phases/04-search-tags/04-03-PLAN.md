  <action>
Create `app/src/components/SearchBar.tsx` as a controlled component:

**Props:**
```typescript
type SearchBarProps = {
  searchText: string
  onSearchTextChange: (text: string) => void
  includeTags: string[]
  onIncludeTagsChange: (tags: string[]) => void
  excludeTags: string[]
  onExcludeTagsChange: (tags: string[]) => void
  placeholder?: string
}
```

**Behavior:**
1. Display filter pills BEFORE text input cursor: `[work] [personal] | <cursor>`
2. Include tags render as light pills, exclude tags render with `-` prefix and different color
3. Clicking X on pill removes it from respective array
4. Backspace when input is empty removes last include tag (if any), else last exclude tag
5. Typing `#tagname` in text input creates include tag pill (support inline autocomplete later in Plan 04)
6. Typing `#-tagname` creates exclude tag pill
7. Text input is controlled (synchronous onChange with `e.target.value`)
8. Use `useRef<HTMLInputElement>` for focus management (exposed via forwardRef if needed)

**Structure:**
```tsx
<div className="searchBar">
  {includeTags.map(tag => (
    <span key={tag} className="filterPill filterPillInclude">
      {tag}
      <button onClick={() => removeIncludeTag(tag)}>×</button>
    </span>
  ))}
  {excludeTags.map(tag => (
    <span key={tag} className="filterPill filterPillExclude">
      -{tag}
      <button onClick={() => removeExcludeTag(tag)}>×</button>
    </span>
  ))}
  <input
    type="text"
    value={searchText}
    onChange={(e) => onSearchTextChange(e.target.value)}
    onKeyDown={handleKeyDown}
    placeholder={includeTags.length || excludeTags.length ? '' : placeholder}
  />
</div>
```

**KeyDown handler:**
- Backspace when input is empty: remove last tag pill
- Enter: do nothing (search is real-time, no submit needed)
- Detect `#` prefix and create tag pills (basic version, full autocomplete in Plan 04)

3. Filter notes using useMemo:

```typescript
}, [notes, searchText, includeTags, excludeTags])
```

4. Replace `notes` with `filteredNotes` in the render (note list map, length checks, etc.)

5. Add Cmd+F keyboard shortcut:
```typescript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'f') {
      e.preventDefault()
      searchInputRef.current?.focus()
    }
  }
  window.addEventListener('keydown', handleKeyDown)
  return () => window.removeEventListener('keydown', handleKeyDown)
}, [])
```

6. Add SearchBar above notes list (after notesHeader div, before loading/notes conditional):
```tsx
<SearchBar
  searchText={searchText}
  onSearchTextChange={setSearchText}
  includeTags={includeTags}
  onIncludeTagsChange={setIncludeTags}
  excludeTags={excludeTags}
  onExcludeTagsChange={setExcludeTags}
  placeholder="Search notes (Cmd+F)"
/>
```

7. Wire tag click in note rows to add include filter:
```typescript
// In the tag map:
onClick={(e) => {
  e.stopPropagation() // Don't trigger row click
  if (e.metaKey || e.ctrlKey) {
    // Exclude tag
    setExcludeTags(prev => prev.includes(tag) ? prev : [...prev, tag])
  } else {
    // Include tag
    setIncludeTags(prev => prev.includes(tag) ? prev : [...prev, tag])
  }
}}
```
