<task type="auto">
  <name>Task 2: Implement WorkspaceService (validate/switch/assign + root-only note listing)</name>
  <files>
app/src-tauri/src/services/workspace_service.rs
app/src-tauri/src/domain/workspace.rs
  </files>
  <action>
- Create `WorkspaceService` that owns an `AppConfigRepo` and provides:
  - `get_state()` -> returns current config plus a computed status for ONLY the active slot:
    - `unassigned` if active slot path is None
    - `ok` if folder exists and is readable
    - `missing` if path does not exist
    - `unreadable` if exists but cannot be read/listed
    - Also compute `fallback_slot: Option<u8>` = first assigned slot with a valid folder (do NOT proactively validate all; only compute fallback by checking until first valid is found).
  - `assign_slot(slot: u8, selected_path: String)`:
    - Canonicalize path if possible (`std::fs::canonicalize`), store as string.
    - Set `active_slot = slot`.
    - Persist and return updated state.
  - `switch_slot(slot: u8)`:
    - If `slot == active_slot`, treat as no-op success.
    - If unassigned, return a typed error `UnassignedSlot { slot }`.
    - If assigned but missing/unreadable, return typed error `InvalidWorkspace { slot, kind }`.
    - If valid, set active_slot, persist, return updated state.
  - `list_root_notes()`:
    - Uses the currently active workspace root.
    - Lists only files directly in the root directory with `.md` extension.
    - Returns a sorted list of basenames (strings), NOT full paths.
    - Does not recurse.

Implementation notes:
- Encode slot <-> index conversion in one place.
- Validation should attempt `read_dir` to detect unreadable.
- Root-only scan naturally excludes `.ponder/` and `deleted/` (directories), but keep the code intentionally root-only.
  </action>
  <verify>
- Add a temporary Rust unit test for the slot index conversion helper (optional if you already ensure correctness via code review).
- `nix develop -c pnpm -C app tauri dev` compiles.
  </verify>
  <done>
- Workspace paths are validated at startup and switch-time; root-only `.md` listing is implemented.
  </done>
</task>
